# Smiley-PCB
This is a simple circuit the starts of frowny :(
but when you push a button, it smiles :)

Electronic Hardware Needed:
- x10 White LEDs (5mm)
- x9 330Ω Resistors
- x2 10kΩ Resistors
- x1 1kΩ Resistor
- x2 Push Buttons
- x1 555 Timer IC (NE555P)
- x1 22nF Non-Polarised Capacitor
- x2 NPN Transistors (2N2222 A331)
